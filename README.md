# Site institucional da PIEMIL

Site estático, responsivo e pronto para publicação.

## Como visualizar

Abra `index.html` no navegador ou publique toda esta pasta em qualquer hospedagem
estática.

## Configurar o WhatsApp

Edite `assets/js/main.js` e preencha `SITE_CONFIG.whatsappUrl` com o link completo:

```js
whatsappUrl: "https://wa.me/5519999999999",
```

Enquanto o campo estiver vazio, os botões de WhatsApp exibem uma mensagem informativa.

O Instagram e o YouTube já usam os perfis informados. Como o material não inclui a
URL exata do Facebook, o link atual abre uma busca por "IEM Limeira" e pode ser
substituído no `index.html` assim que a página oficial for confirmada.

## Estrutura

- `index.html`: conteúdo e SEO.
- `assets/css/styles.css`: identidade visual e responsividade.
- `assets/js/main.js`: menu, animações e configurações.
- `assets/images`: logo e fotografias da liderança.
